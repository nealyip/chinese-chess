# 中國象棋 Chinese Chess

一個純前端、單檔案的中國象棋遊戲，支援人機對戰與雙人對戰。

🎮 **線上試玩：** https://nealyip.github.io/chinese-chess/

---

## 功能

- **人機對戰** — AI 控制黑方，玩家執紅先行
- **雙人對戰** — 兩人輪流在同一台電腦對弈
- **四段難度** — 簡單 / 普通 / 困難 / 地獄
- **完整象棋規則** — 將軍、將死、飛將、絆馬腿等悉數實現
- **走法提示** — 點選棋子後顯示合法落點
- **悔棋** — 人機模式下一次悔回兩步（含 AI 的回應）
- **傳統棋盤視覺** — 仿真實棋盤配色（白底紅線、原木棋子）

## 技術細節

| 項目 | 說明 |
|------|------|
| 實作 | 單一 HTML 檔案，內嵌 CSS + Canvas + JavaScript |
| AI 演算法 | Minimax + Alpha-Beta Pruning |
| 搜索深度 | 簡單 2 / 普通 3 / 困難 4 / 地獄 5 層 |
| 評估函數 | 子力值 + 7 種棋子位置權重表 |
| 走法排序 | 吃子優先（提升剪枝效率） |
| 依賴 | 無任何外部 library，零依賴 |

## 本地運行

直接用瀏覽器開啟 `index.html` 即可，無需任何伺服器。

```bash
open index.html
```

## 棋子走法

| 棋子 | 走法 |
|------|------|
| 將／帥 | 九宮格內直行直列，一步 |
| 士／仕 | 九宮格內斜走，一步 |
| 象／相 | 走「田」字對角，不可過河，象眼可被塞 |
| 馬／傌 | 走「日」字，馬腿可被絆 |
| 車／俥 | 直行直列，不限步數 |
| 炮／砲 | 移動同車，吃子須隔一子 |
| 卒／兵 | 未過河只能前進，過河可左右移動 |
